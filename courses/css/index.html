Lesson 1: Introduction to HTML and CSS

Objective

By the end of this lesson, you will understand the basic structure of an HTML document, the purpose of CSS, and how to apply simple styles to your web pages.

1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures the content on the web, such as text, images, and links.

Basic Structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>

Explanation of Tags:

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the document.
  • <title>: Sets the title of the page (shown in the browser tab).
  • <body>: Contains the visible content of the web page.
  • <h1> to <h6>: Header tags, with <h1> being the highest level.
  • <p>: Paragraph tag for text content.

2. What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance of web pages.

Basic Syntax:


selector {
  property: value;
}

Example:


h1 {
  color: blue;
  font-size: 24px;
}

3. Applying CSS to HTML

Inline CSS:

Styles applied directly within HTML elements using the style attribute.


<h1 style="color: blue; font-size: 24px;">Welcome to My Website</h1>

Pros: Quick and easy for small changes.

Cons: Not reusable, can clutter HTML code.

Internal CSS:

Styles defined within the <style> tag inside the <head> section of an HTML document.


<head>
  <style>
    h1 {
      color: blue;
      font-size: 24px;
    }
    p {
      color: gray;
      font-size: 16px;
    }
  </style>
</head>

Pros: Keeps styles separate from content, easier to manage than inline.

Cons: Styles are confined to a single HTML document.

External CSS:

Styles defined in an external .css file and linked to the HTML document using the <link> tag.


<head>
  <link rel="stylesheet" href="styles.css">
</head>
                

/* styles.css */
h1 {
  color: blue;
  font-size: 24px;
}
p {
  color: gray;
  font-size: 16px;
}

Pros: Reusable across multiple HTML documents, keeps HTML clean, easier maintenance.

Cons: Requires additional HTTP request (though mitigated with caching).

4. Hands-On Exercise: Creating Your First Web Page

  1. Step 1: Open a text editor (e.g., Notepad, VS Code).
  2. Step 2: Create a new file named index.html and add the basic HTML structure. 
    
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
    <style>
      h1 {
        color: blue;
        font-size: 24px;
      }
      p {
        color: gray;
        font-size: 16px;
      }
    </style>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>
  3. Step 3: Add a header (<h1>) and a paragraph (<p>) to the <body>.
  4. Step 4: Apply internal CSS to style the header and paragraph.
  5. Step 5: Save the file and open it in a web browser to see your styled web page.

Additional Tips for Success

  • Practice Regularly: The more you practice writing HTML and CSS, the more comfortable you'll become.
  • Experiment with Styles: Don’t be afraid to try different colors, fonts, and layouts to see what works best.
  • Use Developer Tools: Familiarize yourself with browser developer tools (like Chrome DevTools) to inspect and debug your code.
  • Join Communities: Engage with online communities (such as Stack Overflow, Reddit’s r/webdev, or freeCodeCamp) to ask questions and share knowledge.
  • Build Projects: Apply what you’ve learned by building real projects. This reinforces your skills and builds your portfolio.
